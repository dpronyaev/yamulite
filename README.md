# YAMULITE

Лёгкий кроссплатформенный десктопный клиент для Яндекс Музыки на Python + PyQt6.

## Возможности

- Поиск по трекам, альбомам и исполнителям
- Проигрывание треков (play/pause/seek/volume, очередь, авто-переход)
- Лайк / анлайк понравившихся композиций
- «Мне нравится», список своих плейлистов и их просмотр
- Открытие альбома и списка треков исполнителя из поиска
- Space — play/pause

## Требования

- Python 3.10+
- Установленный **VLC** (для `python-vlc`)
  - macOS: `brew install --cask vlc`
  - Ubuntu/Debian: `sudo apt install vlc`
  - Windows: https://www.videolan.org/vlc/

## Установка

```bash
python -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## Запуск

```bash
python -m yamulite
```

## Авторизация

API Яндекс Музыки больше не принимает пару логин/пароль напрямую, поэтому приложение использует **OAuth device flow**:

1. При первом запуске откроется окно входа.
2. Нажмите «Получить код» — приложение запросит у Яндекса короткий код.
3. Нажмите «Открыть страницу подтверждения» — откроется `https://ya.ru/device`.
4. Войдите на этой странице в свой аккаунт Яндекса и введите показанный код.
5. После подтверждения приложение автоматически получит OAuth-токен и сохранит его в системный keyring (или в `~/.yamulite/token.json`, если keyring недоступен).

Альтернатива — вкладка «Ввести токен» на экране входа: можно вставить уже полученный OAuth-токен напрямую.

Выйти из аккаунта можно через меню «Файл → Выйти из аккаунта» — токен будет удалён.

## Структура

```
yamulite/
  __main__.py          # точка входа
  auth.py              # device-flow + хранение токена
  api.py               # обёртка над yandex_music.Client
  player.py            # VLC-плеер с очередью
  workers.py           # фоновые задачи для Qt
  ui/
    login.py           # экран авторизации
    main_window.py     # главное окно, sidebar, stacked pages
    search.py          # поиск (треки / альбомы / исполнители)
    library.py         # «Мне нравится» и плейлисты
    detail.py          # просмотр альбома / артиста / плейлиста
    player_bar.py      # нижняя панель плеера
    widgets.py         # переиспользуемые виджеты
```
